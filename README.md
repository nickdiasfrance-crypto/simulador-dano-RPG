# Simulador de Dano de RPG

Este projeto é uma ferramenta de simulação em Python para calcular a média e a distribuição de dano de armas em sistemas de RPG (como D&D).

##  Como funciona?
O código utiliza a biblioteca `random` para simular milhares de ataques e a `matplotlib` para gerar um histograma, permitindo visualizar a consistência do dano de diferentes builds.

##  Como executar
1. Certifique-se de ter o Python instalado.
2. Instale as bibliotecas necessárias:
   ```bash
   pip install matplotlib
